============================= README ===================================

Software Name : Pattern_Advisor

		Introduction
	       ==============

Our Web-software i.e Pattern Advisor make tyre loading problem easier.
It fills different types of tyres in an optimised manner for a given amount of volume.

		Description
	       =============

Algorithm (HTML File) : algo.html

Put dimensions of tyres in textarea provided as {width}x{width}x{quantity}
and select the required truck and software will automatically generate the
pictorial view of respective truck as top view.
